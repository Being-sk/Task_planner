Minimal fullstack prototype: React frontend (Vite) + Express backend with a mock AI plan generator.

Quick start

1. Backend

```bash
cd backend
npm install
npm start
```

2. Frontend

```bash
cd frontend
npm install
npm run dev
```

Open `http://localhost:5173` (Vite default) and the frontend will call the backend at `http://localhost:4000`.

See backend/README.md and frontend/README.md for details.
